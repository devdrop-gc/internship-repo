# Common Services on Discovered Ports

| Port  | Service         | Typical Use                     |
|-------|-----------------|---------------------------------|
| 22    | SSH             | Remote login                    |
| 53    | DNS             | Name resolution                 |
| 80    | HTTP            | Web server                      |
| 443   | HTTPS           | Secure web                      |
| 135   | MSRPC           | Microsoft Remote Procedure Call |
| 139   | NetBIOS-SSN     | Windows file/printer sharing    |
| 445   | Microsoft-DS    | SMB/CIFS (Direct SMB)           |
| 1900  | UPnP/SSDP       | Device discovery                |
| 5900  | VNC             | Remote desktop                  |
| 8001  | http-alt        | Custom web/media                |
| 9100  | JetDirect       | Printer service                 |
