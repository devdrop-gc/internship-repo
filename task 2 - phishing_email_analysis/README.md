# Phishing Email Analysis

**Objective:** Identify phishing characteristics in a suspicious email sample.
**Tools Used:** MX Toolbox

## What I Did
1. Used a **real phishing email** with authentic headers (from educational source).
2. Extracted and analyzed **sender spoofing**, **header path**, **IP geolocation**.
3. Identified **grammar errors**, **urgency tactics**, and **social engineering**.
4. Ran headers through **MX Toolbox** and documented results.

## Repository Contents
- `sample_email.txt` – Raw phishing email  
- `email_headers.txt` – Headers extracted for analysis  
- `analysis_report.md` – Step-by-step phishing indicators  
- `screenshots/` – Header analyzer Tool output 

**Sample Source:** Nebraska GenCyber Phishing Module (educational use only).
